The Boxing Challenge

leerdoel:

Je leert in deze oefening hoe je gebruik maakt van verschillende CSS eigenschappen zoals, display: inline-block, float en pseudo-classes. Ook krijg je meer inzicht in het gebruik maken van marges, padding en box-shadow rondom HTML elementen.

Opdracht:

Bekijk de volgende voorbeeld afbeelding en bouw deze zo goed mogelijk na:

Let op de volgende punten:

  • Je mag de CSS toevoegen aan de externe stylesheet css/styles.css
  • Eventueel kun ook je de styling toevoegen binnen de HTML binnen de <style> tags in de <head>
  • Wanneer je over een tekst-box heen beweegt met de muis, dient de achtergrond rood te worden
  • Het kan nodig zijn om in de HTML elementen class attributen toe te voegen en deze te gebruiken in je stylesheet
  • Probeer de boxen gecentreerd binnen je browser te tonen
  • Afhankelijk van het formaat van je browser en scherm-resolutie van je computer kan het voorbeeld iets afwijken, dit is geen probleem
